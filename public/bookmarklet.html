About this Bookmarklet

This bookmarklet works on any page with a <video> element and jQuery. When activated it will make it so the video will pause when you blur from the browser and will continue to play when you refocus to the window. I created this for watching Treehouse videos, so that I can tab out to my code, and automatically have the video pause, and resume when I come back

This is a real bookmarklet I created for my own personal use, but I leave it as an example for all. You can learn about creating bookmarlets on Think Vitamin
